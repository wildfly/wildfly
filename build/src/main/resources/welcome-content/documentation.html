Documentation

Getting Started Guide Introduces you to running the server, how to run the server in standalone and domain modes, how to create a data source, and how to use the Web Management Interface and Command Line interface to manage the server.

Getting Started Developing Applications Introduces you to writing simple Java EE 6 applications and deploying them to the application server. Each tutorial is accompanied by a quickstart sample application. Finally the guide shows you how to create your own application skeleton.

Admin Guide A complete guide to configuring and administrating the application server.

Developer Reference Guide A complete guide to all deployment descriptors and APIs available in the application server.
