SYNOPSIS

    run-batch [--headers={operation_header (;operation_header)*}]

DESCRIPTION

    Runs the currently active batch. The command must be invoked while
    in the batch mode. Executed batch will be discarded and the CLI
    will exit the batch mode.

ARGUMENTS

 --headers  - a list of operation headers separated by a semicolon. For the list of supported
              headers, please, refer to the domain management documentation or use tab-completion.
